# Project Screenshots

Drop your screenshot images into this folder using the exact filenames below.

| File name                  | Project                  |
|----------------------------|--------------------------|
| `it155mp.png`              | IT155MP                  |
| `juresto.png`              | Juresto                  |
| `twtclone.png`             | Twitter Clone Main Page  |
| `it114-expense-tracker.png`| IT114 Expense Tracker    |

Once you drop an image in, both the card thumbnail and the "View Screenshot" link will automatically use it.
